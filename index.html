✉ Email Integration (EmailJS)

Send proposals and invoices with PDF attachments directly from the app. Free: 200 emails/month.

Setup Instructions
  1. Go to emailjs.com and create a free account
  2. Add an Email Service (Gmail, Outlook, or SMTP)
  3. Create a Template with these variables:
    {{to_email}} {{to_name}} {{from_name}} {{reply_to}} {{subject}} {{message}}
  4. In Template Settings > Attachments, add a Dynamic Attachment with variable name attachment
  5. Copy your Public Key (Account > API Keys), Service ID, and Template ID into the fields above
